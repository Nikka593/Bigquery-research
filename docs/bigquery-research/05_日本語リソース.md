## 1. 最新機能・技術動向

### Google Cloud Next '25での発表

2025年に開催された Google Cloud Next '25 では、BigQuery に関する大型アップデートが複数発表された。

| 機能 | 概要 |
|---|---|
| **BigQuery Data Agents** | Data Engineering Agent / Data Governance Agent / Data Science Agent / Conversational Analytics Agent の4種類のAIエージェントが登場 |
| **Conversational Analytics** | 自然言語でBigQueryデータに問い合わせ可能なPublic Preview機能。SQL不要でグラフや表を生成 |
| **BigQuery グローバルクエリ** | 単一SQLで複数リージョンにまたがる分散データを横断検索。ETL不要のゼロETLエクスペリエンスを実現 |
| **自律型エンベディング生成** | `GENERATED ALWAYS AS AI.EMBED(...)` 構文でエンベディングを自動管理。`AI.SEARCH()` による類似検索が可能に |
| **BigQuery Universal Catalog** | Dataplex Catalog がリネーム。ガバナンス機能を統合強化 |

### Gemini in BigQuery の現状 (2025年)

- **SQL / Python コード生成**: Geminiによる自然言語からのSQL自動生成。2025年5月25日以降、全エディションで追加費用なしで利用可能
- **Data Canvas (データキャンバス)**: 2025年4月7日にGA (一般提供)。自然言語でグラフやデータ分析を自動生成
- **日本語サポート**: Conversational Analytics は日本語プロンプトに対応。Data Canvas は日本語でも動作確認済み

## 3. 日本企業の活用事例

### DMM.com - リアルタイムレコメンドシステム

**概要:** DMM.comのデータ基盤開発部が、BigQuery継続的クエリとVertex AIを組み合わせたリアルタイムレコメンドシステムを構築。

**技術スタック:**
- Google Analytics 4 (GA4) → BigQuery 自動連携
- BigQuery 継続的クエリ (Continuous Queries) でリアルタイムストリーム処理
- Bigtable / Pub/Sub へのリアルタイムエクスポート
- Vertex AI による機械学習モデルの推論

**発表:** Google Cloud Next Tokyo 2025 にてセッション登壇

---

### ZOZO - 大規模データ基盤の運用

**概要:** ZOZOでは2025年2月時点で1,100超のデータマートをBigQuery上で運用。dbt CoreとCloud Composerを組み合わせた堅牢なデータパイプラインを構築。

**特徴:**
- BigQuery + dbt Core による変換管理
- Apache Airflow (Cloud Composer) によるジョブスケジューリング
- ストレージをライフサイクルと課金モデルの2軸8種類に分類してコスト最適化

---

### DeNA - BigQuery Editions移行でコスト70%削減

**概要:** DeNAのゲームエンタメグループが2025年2月にBigQuery Editionsへ移行完了。複数プロジェクトで47〜70%のコスト削減を達成。

---

### 有隣堂 - 老舗書店チェーンのDXとデータ可視化

**概要:** 創業200年超の老舗書店チェーン・株式会社有隣堂が、クラウドエースの支援によりBigQueryを導入。社内データの可視化とデータドリブン経営を推進。
