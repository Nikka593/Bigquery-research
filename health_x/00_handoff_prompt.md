## 決定済み：情報収集プロセス

毎日以下のプロンプトでClaude（またはClaude in Chrome）を使い最新情報を収集：

```
今日（YYYY年MM月DD日）に発表・話題になっている健康・医学の最新研究を5つ教えてください。
条件：
1. 査読済み論文または権威ある機関（NIH/WHO/Harvard/Stanford/Nature/NEJM等）の発表
2. 各研究に：内容（1文）・重要な数値・実生活での応用方法・ソース
3. カテゴリ：栄養/睡眠/運動/メンタル/長寿からバランスよく
```

収集した5件から**スコアリング**で上位3件を投稿：
- 新しさ30点・実用性25点・意外性20点・信頼性15点・汎用性10点
- 70点以上のみ投稿候補

## 決定済み：自動化システム

GitHubリポジトリに以下のPythonスクリプトが作成済み（`health_x/03_automation/`）：

- `research_fetcher.py`：Claude API（claude-opus-4-6）で研究収集→ツイート文生成
- `twitter_poster.py`：Tweepy v4でX APIに投稿（DRY_RUNモード対応）
- `scheduler.py`：毎日6時に生成、7時/12時/21時に自動投稿
- `manual_post.py`：レビュー・編集・承認してから投稿する手動モード

**実行に必要なもの：**
- Anthropic API キー
- X Developer API（Basic以上: $100/月）
