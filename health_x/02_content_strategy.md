## 情報収集プロセス（毎日の作業フロー）

### 使用する情報源（信頼度A）

**学術・研究機関：**
- PubMed (pubmed.ncbi.nlm.nih.gov)
- Nature.com
- Cell.com
- NEJM (nejm.org)
- The Lancet

**信頼できる科学メディア：**
- ScienceDaily (sciencedaily.com)
- Examine.com（栄養・サプリ専門）
- Huberman Lab（スタンフォード神経科学）
- Peter Attia MD（長寿・パフォーマンス）

### Claude in Chrome での情報収集手順

```
毎日の情報収集プロンプト：

「今日（YYYY年MM月DD日）に発表された、または話題になっている
健康・医学に関する最新研究を5つ教えてください。
以下の条件で：
1. 査読済み論文または権威ある機関の発表のみ
2. 各研究に：
   - 研究内容（1文）
   - 重要な数値データ
   - 実生活での応用方法
   - ソース（論文名、発表機関、年）
3. カテゴリ：栄養/睡眠/運動/メンタル/長寿からバランスよく」
```

### 珠玉3選の選定基準

投稿する3つを選ぶ基準（スコアリング）：

| 基準 | 配点 |
|------|------|
| 新しさ（過去30日以内） | 30点 |
| 実用性（明日からできる） | 25点 |
| 驚き・意外性 | 20点 |
| データの確かさ（査読済み等） | 15点 |
| 広い層への関連性 | 10点 |

合計70点以上の研究を投稿候補とし、上位3つを選択。

## 投稿品質チェックリスト

投稿前に必ず確認：

- [ ] ソース（論文名・発表機関・年）が明記されている
- [ ] 誇張していない（「治る」「治療」などの医療的表現を避ける）
- [ ] 具体的な数値・データが含まれている
- [ ] 「実際にどうする？」が含まれている
- [ ] 日本語と英語の両方が含まれている（または交互投稿）
- [ ] ハッシュタグが適切（3-5個）
- [ ] 240文字以内（または適切なスレッド形式）
- [ ] 免責事項：医療アドバイスではない旨（月1回程度）

## エンゲージメント戦略

### リプライ戦略
- 投稿後1時間以内にコメントへ返信（アルゴリズム優遇）
- 「あなたはどう思いますか？」「試したことありますか？」を活用
- フォロワーの質問をネタに翌日の投稿を作成

### フォロー戦略
- 健康・科学系アカウント（フォロワー1万以上）のフォロワーをフォロー
- 1日50-100人を上限にフォロー
- フォローバックされない場合は7日後にアンフォロー

### ハッシュタグ戦略
- 大きすぎるタグ（#health = 競争激しすぎ）は避ける
- ニッチタグ（#SleepScience, #NutritionResearch）でリーチ
- 日英両方のタグを必ず入れる

### コラボ・引用戦略
- 有名健康系アカウント（@hubermanlab等）の投稿を引用ツイート
- 「これに追加で：〇〇という研究もあります」スタイル
- 月2-3回、他の健康系アカウントと相互コラボ投稿
