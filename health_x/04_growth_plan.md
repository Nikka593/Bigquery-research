## フェーズ別戦略（30日間）

### Phase 1：基盤構築（Day 1-7）
**目標：フォロワー0 → 500人**

**やること：**
1. アカウント設定完了（プロフィール・ヘッダー・アイコン）
2. ピン留めツイートを投稿
3. 過去ツイートを全削除
4. 初週は**投稿品質＞量**：1日3投稿を完璧に
5. 健康・科学系の大型アカウント100個をフォロー
6. フォローした人のツイートに有益な返信を10回/日

**初週に狙うバズ戦略：**
- 「10の科学的健康事実【保存推奨】」スレッドを1本投稿
- 有名健康系アカウントの投稿を引用ツイート（有益な追加情報付きで）

---

### Phase 2：拡散エンジン構築（Day 8-14）
**目標：フォロワー500 → 2,000人**

**やること：**
1. **スレッド投稿を週2本**に増やす（スレッドはリーチが3-5倍）
2. 「逆張りコンテンツ」を週1本：「科学的に否定された健康常識5選」
3. ナイトルーティン系コンテンツは特に拡散しやすい（夜の投稿に注力）
4. 返信欄でのコミュニティ形成：質問を積極的に投げかける
5. 同規模（500-5000人）の健康アカウントと相互フォロー・コラボ

**Week 2のキラーコンテンツ案：**
```
「実は逆効果だった習慣5選【科学的根拠あり】」
「睡眠の質を上げる科学的な方法TOP5」
「コーヒーについて科学が言っていること全部まとめ」
```

---

### Phase 3：バズサイクル確立（Day 15-21）
**目標：フォロワー2,000 → 5,000人**

**やること：**
1. **バイリンガル効果の最大化**：英語圏と日本語圏の両方でトレンド乗り
2. 日本で話題の健康ニュースを英語でも発信（日→英ブリッジ戦略）
3. 「今週の科学的健康Tips 5選」を毎週日曜に定番コンテンツ化
4. インフルエンサー（フォロワー10万以上）の投稿への高品質返信

**拡散公式：**
- ショッキングな数字 + シンプルな解決策 + ソース = バイラルコンテンツ
- 例：「砂糖を1日30g減らすだけで、3ヶ月後の認知機能が14%改善（Nature誌）」

---

### Phase 4：加速・維持（Day 22-30）
**目標：フォロワー5,000 → 10,000人**

**やること：**
1. **コンテンツのデータ分析**：この3週間で最もインプレッションが高かった投稿のパターンを分析し、同じフォーマットを量産
2. **コラボ投稿**：同規模〜大型アカウントとの対談形式ツイート
3. 毎週月曜に「今週の健康目標」投票を実施（エンゲージメント急増）
4. フォロワーの質問に答える「Q&A Sunday」定番化

## バイラルコンテンツの設計原則

### 拡散されるツイートの共通要素
1. **数字がある**：「47%」「3週間で」「〇〇の研究で」
2. **意外性がある**：予想を裏切る事実
3. **短時間で読める**：スキャンして理解できる構造
4. **保存したくなる**：後で使える実用情報
5. **シェアしたくなる**：「友達にも教えたい」感

### NGパターン（避けること）
- 医療アドバイス的な表現（「治る」「改善される」断定）
- ソースなしの情報
- ネガティブ・恐怖を煽るコンテンツ
- 長すぎて読む気がなくなる投稿

## アルゴリズム対策

### X（旧Twitter）アルゴリズムで優遇される行動
1. **投稿後30分が勝負**：この間にいいね・RT・返信が多いほど拡散
2. **外部リンクは最小限**：リンクありの投稿はリーチが下がる傾向
3. **返信への返信**：コメントへの返信はエンゲージメントシグナル
4. **スレッド投稿**：滞在時間が長くなるため優遇
5. **投稿の一貫性**：毎日同じ時間に投稿するアカウントは優遇

## リスクと対策

| リスク | 対策 |
|--------|------|
| 情報が不正確だった場合 | 必ずソースを明記。誤情報は即座に訂正ツイート |
| 投稿が続かない | 自動化スクリプトで負担軽減。週1日は休日OK |
| シャドウバン | スパム行為を避ける（1日50フォロー上限遵守） |
| X API制限 | 手動投稿のバックアップフローを用意 |
| 医療的トラブル | プロフィールに「医療アドバイスではありません」明記 |
